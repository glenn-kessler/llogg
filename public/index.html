About Life Logger

Version

Personal life tracking application for logging daily activities, moods, and events.

Key Features

Core Functionality

  • Quick logging with customizable types and details
  • Automatic timestamp capture with adjustment options
  • Automatic GPS location tracking (~10m accuracy)
  • IP-based location fallback when GPS unavailable
  • IndexedDB storage for offline-first functionality
  • PWA support for mobile installation

Visualization & Analysis

  • Time-based aggregation with flexible step sizes (1h, 6h, 1 day, 1 week, 1 month)
  • Dual-axis chart modes: Y-axis (time-based) and X-axis (item-based)
  • Multiple chart types: Bar, Line, Pie
  • Grouped bar charts for multiple items per time step
  • Multi-line charts with distinct colors per item
  • Instant filter updates without Apply button
  • Customizable time spans and aggregation levels

Data Management

  • CSV export/import with full data preservation
  • Human-readable configuration export/import (types and details)
  • Configuration saved as indented text format for easy editing
  • Location data included in CSV exports
  • Bulk operations: Move and delete multiple details
  • Drag-and-drop reordering of details
  • Custom icons and colors for types and details
  • Custom units per entry (flexible measurement)

User Experience

  • Mobile-first responsive design
  • Touch-optimized UI (48px minimum targets)
  • Last-used preferences automatically restored
  • LocalStorage persistence for view settings
  • Configuration mode for advanced editing
  • Visual feedback with pulsing animations

Built with Vanilla JavaScript • No external dependencies • Privacy-first
